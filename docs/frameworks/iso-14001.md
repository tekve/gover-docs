# ISO 14001 Compliance with Gover

ISO 14001 is the international standard for environmental management systems.

## Overview

| Attribute | Value |
|-----------|-------|
| **Full Name** | ISO 14001:2015 |
| **Type** | International Standard |
| **Publisher** | ISO |
| **Focus** | Environmental management |

## Purpose

ISO 14001 helps organizations:

- Manage environmental responsibilities
- Achieve environmental objectives
- Meet compliance obligations
- Improve the level of environmental protection

## Structure

| Clause | Topic |
|--------|-------|
| **4** | Context of the organization |
| **5** | Leadership |
| **6** | Planning |
| **7** | Support |
| **8** | Operation |
| **9** | Performance evaluation |
| **10** | Improvement |

## Key Requirements

### Environmental Aspects

- Identify environmental aspects
- Determine significant impacts
- Lifecycle perspective

### Compliance Obligations

- Legal requirements
- Other requirements
- Compliance evaluation

### Operational Control

- Processes to meet requirements
- Emergency preparedness
- Outsourced processes

## Using Gover for ISO 14001

### 1. Add the Framework

1. Go to **Frameworks** → **Add Framework**
2. Select **ISO 14001** from templates
3. Add to your workspace

### 2. Identify Environmental Aspects

Use Gover's risk management:

1. Create risks for environmental aspects
2. Assess significance
3. Determine required controls

### 3. Map Controls

Map controls for:

- Environmental policies
- Operating procedures
- Monitoring programs
- Emergency response

### 4. Track Performance

Monitor:

- Environmental objectives
- Compliance status
- Audit findings
- Improvement actions

## Integration

ISO 14001 integrates with:

- **ISO 9001** — Quality
- **ISO 45001** — Occupational health and safety
- **ISO 27001** — Information security
