# ISO 9001 Compliance with Gover

ISO 9001 is the international standard for quality management systems (QMS).

## Overview

| Attribute | Value |
|-----------|-------|
| **Full Name** | ISO 9001:2015 |
| **Type** | International Standard |
| **Publisher** | ISO |
| **Focus** | Quality management |

## Purpose

ISO 9001 helps organizations:

- Consistently produce high-quality products/services
- Improve customer satisfaction
- Address risks and opportunities
- Demonstrate conformity to requirements

## Structure

| Clause | Topic |
|--------|-------|
| **4** | Context of the organization |
| **5** | Leadership |
| **6** | Planning |
| **7** | Support |
| **8** | Operation |
| **9** | Performance evaluation |
| **10** | Improvement |

## Key Principles

1. Customer focus
2. Leadership
3. Engagement of people
4. Process approach
5. Improvement
6. Evidence-based decision making
7. Relationship management

## Using Gover for ISO 9001

### 1. Add the Framework

1. Go to **Frameworks** → **Add Framework**
2. Select **ISO 9001** from templates
3. Add to your workspace

### 2. Document Processes

Map your quality processes:

- Core processes
- Support processes
- Management processes

### 3. Map Controls

Link controls for:

- Quality policies
- Process controls
- Document control
- Corrective actions

### 4. Track Performance

Monitor:

- Quality objectives
- Customer satisfaction
- Process performance
- Audit results

## Integration

ISO 9001 integrates with:

- **ISO 14001** — Environmental
- **ISO 45001** — Occupational health and safety
- **ISO 27001** — Information security
