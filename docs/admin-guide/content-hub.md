# Content Hub

Manage templates, discover content, and publish to the marketplace.

## What is Content Hub?

Content Hub is your central place for:

- **Templates** — Pre-built frameworks, controls, and documents
- **Discovery** — Browse marketplace content
- **Installed** — Content you've added to your workspace

## Accessing Content Hub

Navigate to **Organization Settings** → **Content Hub**

## Templates

### Using Templates

1. Go to **Templates**
2. Browse available templates
3. Click **Use Template**
4. Select target workspace
5. Customize as needed

### Template Types

| Type | Description |
|------|-------------|
| **Frameworks** | Complete framework structures |
| **Controls** | Control libraries |
| **Documents** | Document templates |
| **Risk Templates** | Pre-defined risks |

## Discovery

### Browsing the Marketplace

1. Go to **Discovery**
2. Browse by category:
   - Frameworks
   - Controls
   - Documents
3. Use search and filters
4. Preview content

### Installing Content

1. Find content you want
2. Click **Install**
3. Select target workspace
4. Content is added to your workspace

### Content Details

Before installing, review:

- Description
- Publisher
- Ratings and reviews
- What's included

## Installed Content

### Viewing Installed Content

See what you've installed:

- Source (marketplace/template)
- Installation date
- Workspace

### Managing Installed Content

Installed content becomes yours to customize:

- Edit as needed
- No automatic updates
- Delete if not needed

## Publishing Best Practices

1. **Quality content** — Ensure content is complete and accurate
2. **Good descriptions** — Help others understand what's included
3. **Appropriate tags** — Make content discoverable
4. **Keep updated** — Maintain published content

## Content Packs

Bundle related content together:

### Creating a Pack

1. Go to **My Content**
2. Click **Create Pack**
3. Add items:
   - Frameworks
   - Controls
   - Documents
4. Add pack metadata
5. Publish

### Installing Packs

Install entire packs at once:

1. Find pack in Discovery
2. Click **Install Pack**
3. All items are added together
