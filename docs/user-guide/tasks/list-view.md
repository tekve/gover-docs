# Task List View

Manage tasks in a traditional table format.

## Using List View

Navigate to **Tasks** → **All Tasks**

## Table Columns

| Column | Description |
|--------|-------------|
| **Title** | Task name (click to open) |
| **Status** | Current status |
| **Assignee** | Person responsible |
| **Due Date** | Deadline |
| **Priority** | High, Medium, Low |
| **Category** | Task category |
| **Created** | Creation date |

## Sorting

Click column headers to sort:
- Click once for ascending
- Click again for descending
- Sort by any column

## Filtering

Use the filter bar to narrow results:

### Quick Filters
- **My Tasks** — Tasks assigned to you
- **Overdue** — Past due date
- **Due This Week** — Upcoming deadlines

### Advanced Filters
- Status
- Assignee
- Priority
- Category
- Date range
- Tags

## Bulk Actions

Select multiple tasks for bulk operations:

1. Check boxes next to tasks
2. Click **Bulk Actions**
3. Choose action:
   - Change status
   - Assign to person
   - Set priority
   - Delete

## Creating Tasks

1. Click **Add Task**
2. Fill in details:
   - Title (required)
   - Description
   - Assignee
   - Due date
   - Priority
   - Category
3. Save

## Task Details

Click a task to see full details:

### Overview Tab
- Title and description
- Status and priority
- Assignee and due date

### Related Tab
- Linked controls
- Linked documents
- Linked risk assessments

### Comments Tab
- Team discussions
- Activity log

### History Tab
- Change history
- Audit trail

## Exporting

Export task list:
1. Apply desired filters
2. Click **Export**
3. Choose format (CSV, Excel)
4. Download

## Best Practices

1. **Use filters** — Focus on relevant tasks
2. **Sort by due date** — Prioritize deadlines
3. **Bulk update** — Efficient for multiple tasks
4. **Export regularly** — For reporting and backup
