# Kanban Board

Manage tasks visually with the Kanban board.

## What is Kanban?

Kanban is a visual workflow management method:

- Tasks displayed as cards
- Columns represent statuses
- Drag and drop to update

## Using the Kanban Board

Navigate to **Tasks** → **Kanban**

### Board Layout

```
┌─────────┬─────────┬─────────┬─────────┐
│  Task   │In Prog. │ Review  │  Done   │
├─────────┼─────────┼─────────┼─────────┤
│ [Task]  │ [Task]  │ [Task]  │ [Task]  │
│ [Task]  │ [Task]  │         │ [Task]  │
│ [Task]  │         │         │         │
└─────────┴─────────┴─────────┴─────────┘
```

### Task Cards

Each card shows:

- Task title
- Assignee avatar
- Due date
- Priority indicator
- Category tag

## Moving Tasks

### Drag and Drop

1. Click and hold a task card
2. Drag to the target column
3. Release to update status

### Quick Actions

1. Click the task card
2. Use the status dropdown
3. Select new status

## Creating Tasks

### Quick Create

1. Click **+ Add Task** in any column
2. Enter task title
3. Press Enter
4. Task is created in that status

### Full Create

1. Click **Add Task** button
2. Fill in all details
3. Save

## Filtering the Board

Filter tasks displayed:

- **Assignee** — Show specific person's tasks
- **Priority** — Filter by priority level
- **Category** — Show specific categories
- **Due Date** — Filter by deadline

## Board Customization

### Column Order

Drag columns to reorder them.

### WIP Limits

Set work-in-progress limits per column:

1. Go to **Tasks** → **Settings** → **Statuses**
2. Set limit for each status
3. Board shows warning when exceeded

## Best Practices

1. **Limit WIP** — Don't overload "In Progress"
2. **Update regularly** — Keep the board current
3. **Use filters** — Focus on relevant tasks
4. **Review daily** — Check board in standups
5. **Clear blockers** — Address stuck tasks quickly
