# Uploading Documents

Add files to your Gover document library.

## Supported Formats

| Format | Extensions |
|--------|------------|
| **Documents** | PDF, DOC, DOCX, ODT |
| **Spreadsheets** | XLS, XLSX, CSV |
| **Presentations** | PPT, PPTX |
| **Images** | PNG, JPG, GIF |
| **Text** | TXT, MD |

## Upload Methods

### Single Upload

1. Navigate to **Documents**
2. Click **Add Document**
3. Select **Upload File**
4. Choose your file
5. Fill in metadata:
   - Title
   - Description
   - Category
6. Click **Upload**

### Drag and Drop

1. Open the Documents page
2. Drag files from your computer
3. Drop onto the upload area
4. Complete metadata for each file

### Bulk Upload

1. Click **Add Document** → **Bulk Upload**
2. Select multiple files
3. Review and edit metadata
4. Confirm upload

## Document Metadata

When uploading, provide:

| Field | Required | Description |
|-------|----------|-------------|
| **Title** | Yes | Document name |
| **Description** | No | Summary of contents |
| **Category** | No | Organizational grouping |
| **Tags** | No | Searchable tags |
| **Owner** | No | Responsible person |

## File Size Limits

- Maximum file size: 50 MB
- For larger files, consider linking to external storage

## Version Control

Upload new versions of existing documents:

1. Open the document
2. Click **Upload New Version**
3. Select the updated file
4. Add version notes
5. Confirm

Previous versions are retained in history.

## Best Practices

1. **Use clear names** — Descriptive titles help findability
2. **Add descriptions** — Summarize document contents
3. **Categorize consistently** — Use standard categories
4. **Tag appropriately** — Add relevant tags for search
5. **Version properly** — Upload new versions, don't create duplicates
