# Kanban-taulu

Hallitse tehtäviä visuaalisesti Kanban-taululla.

## Mikä on Kanban?

Kanban on visuaalinen työnkulun hallintamenetelmä:

- Tehtävät näytetään kortteina
- Sarakkeet edustavat tiloja
- Vedä ja pudota päivittääksesi

## Kanban-taulun käyttö

Siirry kohtaan **Tehtävät** → **Kanban**

### Taulun asettelu

```
┌─────────┬─────────┬─────────┬─────────┐
│ Tehtävä │Käynnissä│Katselmoi│ Valmis  │
├─────────┼─────────┼─────────┼─────────┤
│ [Tehtä] │ [Tehtä] │ [Tehtä] │ [Tehtä] │
│ [Tehtä] │ [Tehtä] │         │ [Tehtä] │
│ [Tehtä] │         │         │         │
└─────────┴─────────┴─────────┴─────────┘
```

### Tehtäväkortit

Jokainen kortti näyttää:

- Tehtävän otsikon
- Tekijän avatarin
- Määräpäivän
- Prioriteetti-ilmaisimen
- Kategoriamerkin

## Tehtävien siirtäminen

### Vedä ja pudota

1. Klikkaa ja pidä tehtäväkorttia
2. Vedä kohdesarakkeeseen
3. Päästä irti päivittääksesi tilan

### Pikatoiminnot

1. Klikkaa tehtäväkorttia
2. Käytä tila-alasvetovalikkoa
3. Valitse uusi tila

## Tehtävien luominen

### Pikaluonti

1. Klikkaa **+ Lisää tehtävä** missä tahansa sarakkeessa
2. Syötä tehtävän otsikko
3. Paina Enter
4. Tehtävä luodaan kyseiseen tilaan

### Täysi luonti

1. Klikkaa **Lisää tehtävä** -painiketta
2. Täytä kaikki tiedot
3. Tallenna

## Taulun suodattaminen

Suodata näytettäviä tehtäviä:

- **Tekijä** — Näytä tietyn henkilön tehtävät
- **Prioriteetti** — Suodata prioriteettitason mukaan
- **Kategoria** — Näytä tietyt kategoriat
- **Määräpäivä** — Suodata deadlinen mukaan

## Taulun mukauttaminen

### Sarakkeiden järjestys

Vedä sarakkeita järjestääksesi ne uudelleen.

### WIP-rajoitukset

Aseta työn alla olevien tehtävien rajat per sarake:

1. Mene kohtaan **Tehtävät** → **Asetukset** → **Tilat**
2. Aseta raja kullekin tilalle
3. Taulu näyttää varoituksen ylittyessään

## Parhaat käytännöt

1. **Rajoita WIP** — Älä ylikuormita "Käynnissä"-saraketta
2. **Päivitä säännöllisesti** — Pidä taulu ajan tasalla
3. **Käytä suodattimia** — Keskity relevantteihin tehtäviin
4. **Tarkista päivittäin** — Käy taulu läpi palavereissa
5. **Poista esteet** — Käsittele juuttuneet tehtävät nopeasti
