# Tehtävien luettelonäkymä

Hallitse tehtäviä perinteisessä taulukkomuodossa.

## Luettelonäkymän käyttö

Siirry kohtaan **Tehtävät** → **Kaikki tehtävät**

## Taulukon sarakkeet

| Sarake           | Kuvaus                             |
| ---------------- | ---------------------------------- |
| **Otsikko**      | Tehtävän nimi (klikkaa avataksesi) |
| **Tila**         | Nykyinen tila                      |
| **Tekijä**       | Vastuuhenkilö                      |
| **Määräpäivä**   | Deadline                           |
| **Prioriteetti** | Korkea, Keskitaso, Matala          |
| **Kategoria**    | Tehtävän kategoria                 |
| **Luotu**        | Luontipäivämäärä                   |

## Lajittelu

Klikkaa sarakkeen otsikoita lajitellaksesi:

- Klikkaa kerran nousevaan järjestykseen
- Klikkaa uudelleen laskevaan järjestykseen
- Lajittele minkä tahansa sarakkeen mukaan

## Suodattaminen

Käytä suodatinpalkkia tulosten rajaamiseen:

### Pikasuodattimet

- **Omat tehtävät** — Sinulle osoitetut tehtävät
- **Myöhässä** — Määräpäivä ylitetty
- **Tällä viikolla** — Tulevat deadlinet

### Edistyneet suodattimet

- Tila
- Tekijä
- Prioriteetti
- Kategoria
- Aikaväli
- Tunnisteet

## Massatoiminnot

Valitse useita tehtäviä massatoiminnoille:

1. Valitse tehtävien vieressä olevat valintaruudut
2. Klikkaa **Massatoiminnot**
3. Valitse toiminto:
   - Muuta tilaa
   - Osoita henkilölle
   - Aseta prioriteetti
   - Poista

## Tehtävien luominen

1. Klikkaa **Lisää tehtävä**
2. Täytä tiedot:
   - Otsikko (pakollinen)
   - Kuvaus
   - Tekijä
   - Määräpäivä
   - Prioriteetti
   - Kategoria
3. Tallenna

## Tehtävän tiedot

Klikkaa tehtävää nähdäksesi täydet tiedot:

### Yleiskatsaus-välilehti

- Otsikko ja kuvaus
- Tila ja prioriteetti
- Tekijä ja määräpäivä

### Liittyvät-välilehti

- Yhdistetyt kontrollit
- Yhdistetyt dokumentit
- Yhdistetyt riskinarvioinnit

### Kommentit-välilehti

- Tiimin keskustelut
- Aktiviteettiloki

### Historia-välilehti

- Muutoshistoria
- Tarkastuspolku

## Vienti

Vie tehtävälista:

1. Aseta halutut suodattimet
2. Klikkaa **Vie**
3. Valitse muoto (CSV, Excel)
4. Lataa

## Parhaat käytännöt

1. **Käytä suodattimia** — Keskity relevantteihin tehtäviin
2. **Lajittele määräpäivän mukaan** — Priorisoi deadlinet
3. **Massapäivitys** — Tehokas useille tehtäville
4. **Vie säännöllisesti** — Raportointia ja varmuuskopiointia varten
