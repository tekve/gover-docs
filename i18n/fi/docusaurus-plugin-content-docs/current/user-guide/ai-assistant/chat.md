# AI-chat-käyttöliittymä

Keskustele Goverin AI:n kanssa saadaksesi apua vaatimustenmukaisuustehtäviin.

## Chatin avaaminen

Klikkaa **AI-avustaja** -kuvaketta sivupalkissa avataksesi chat-paneelin.

## Chatin kyvykkyydet

### Kysy kysymyksiä

Kysy vaatimustenmukaisuusdatastasi:

- "Mitkä hallintakeinot on yhdistetty GDPR:n artiklaan 32?"
- "Näytä minulle korkean riskin kohteet"
- "Millä vaatimuksilla ei ole hallintakeinoja?"

### Saa ohjausta

Kysy vaatimustenmukaisuusohjeita:

- "Miten minun tulisi lähestyä ISO 27001 -sertifiointia?"
- "Mitä hallintakeinoja tyypillisesti tarvitaan pääsynhallintaan?"
- "Selitä ero kokonaisriskin ja jäännösriskin välillä"

### Navigoi

Pyydä AI:ta auttamaan asioiden löytämisessä:

- "Vie minut riskirekisteriin"
- "Näytä minulle Pääsynhallintapolitiikka-dokumentti"
- "Avaa ISO 27001 -viitekehys"

### Luo sisältöä

Pyydä AI:ta auttamaan sisällön luonnostelussa:

- "Auta minua kirjoittamaan kuvaus pääsynhallintapolitiikalle"
- "Ehdota riskikuvaus tietomurrolle"

## Chatin käyttö

### Keskustelun aloittaminen

1. Avaa AI-paneeli
2. Kirjoita kysymyksesi tai pyyntösi
3. Paina Enter tai klikkaa Lähetä
4. Tarkista vastaus

### Jatkokysymykset

Jatka keskustelua:

- Kysy tarkentavia kysymyksiä
- Pyydä lisätietoja
- Tarkenna pyyntöäsi

### Chat-historia

Pääse aiempiin keskusteluihin:

1. Klikkaa historia-kuvaketta
2. Valitse aiempi keskustelu
3. Jatka siitä mihin jäit

## Esimerkkikeskusteluja

### Puutteiden löytäminen

```
Sinä: Mitkä ISO 27001 -vaatimukset eivät ole yhdistetty hallintakeinoihin?

AI: Löysin 12 vaatimusta ilman yhdistettyjä hallintakeinoja:
- A.5.1.1 - Tietoturvapolitiikat
- A.5.1.2 - Politiikkojen katselmointi
...
Haluatko, että ehdotan hallintakeinoja näille?
```

### Ehdotusten saaminen

```
Sinä: Ehdota hallintakeinoja A.5.1.1:lle

AI: Vaatimuksen "Tietoturvapolitiikat" perusteella
suosittelen näitä hallintakeinoja:
1. Tietoturvapolitiikka (90% vastaavuus)
2. Tietoturvapolitiikkaviitekehys (85% vastaavuus)
...
```

## Parhaat käytännöt

1. **Ole täsmällinen** — Selkeät kysymykset saavat parempia vastauksia
2. **Anna konteksti** — Mainitse viitekehys tai aihealue
3. **Iteroi** — Tarkenna kysymyksiäsi vastausten perusteella
4. **Varmista** — Tarkista aina AI-ehdotukset ennen soveltamista
