# AI-ehdotukset

Saa älykkäitä suosituksia yhteyksiin ja sisältöön.

## Ehdotustyypit

### Hallintakeino-vaatimus -yhdistäminen

AI ehdottaa mitkä hallintakeinot täyttävät mitkä vaatimukset.

### Dokumentti-hallintakeino -yhdistäminen

AI ehdottaa mitkä dokumentit toimivat todisteina hallintakeinoille.

### Riskien käsittely

AI ehdottaa hallintakeinoja tunnistettujen riskien lieventämiseen.

## Ehdotusten saaminen

### Vaatimuksista

1. Avaa vaatimus
2. Klikkaa **Yhdistä hallintakeinot**
3. Klikkaa **AI-ehdotukset**
4. Tarkista ehdotetut hallintakeinot luottamusluvuineen
5. Hyväksy tai hylkää kukin ehdotus

### Kontrolleista

1. Avaa hallintakeino
2. Klikkaa **Yhdistä vaatimukset** tai **Yhdistä dokumentit**
3. Klikkaa **AI-ehdotukset**
4. Tarkista ja hyväksy ehdotukset

### Dokumenteista

1. Avaa dokumentti
2. Klikkaa **Yhdistä hallintakeinot**
3. Klikkaa **AI-ehdotukset**
4. Tarkista ehdotetut hallintakeinot

## Ehdotusten ymmärtäminen

### Luottamusluvut

Jokainen ehdotus sisältää luottamusluvun:

| Luku     | Merkitys                  |
| -------- | ------------------------- |
| 90-100%  | Erittäin korkea luottamus |
| 70-89%   | Korkea luottamus          |
| 50-69%   | Keskitason luottamus      |
| Alle 50% | Matala luottamus          |

### Perustelut

AI tarjoaa perustelut kullekin ehdotukselle:

- Tekstin samankaltaisuusanalyysi
- Olemassa olevat yhdistämismallit
- Alan parhaat käytännöt

## Ehdotusten käsittely

### Hyväksy

Klikkaa **Hyväksy** luodaksesi yhteyden.

### Hylkää

Klikkaa **Hylkää** ohittaaksesi ehdotuksen.

### Muokkaa

Hyväksy ehdotus, sitten muokkaa yhteyttä tarpeen mukaan.

### Massahyväksy

Valitse useita ehdotuksia ja hyväksy ne kaikki kerralla.

## Ehdotusten laatu

AI-ehdotukset paranevat kun:

- Sinulla on enemmän olemassa olevia yhteyksiä (AI oppii malleja)
- Kuvaukset ovat yksityiskohtaisia ja selkeitä
- Annat palautetta hyväksymällä/hylkäämällä

## Parhaat käytännöt

1. **Tarkista ennen hyväksymistä** — AI ei ole täydellinen
2. **Tarkista luottamusluvut** — Korkeampi on parempi
3. **Lue perustelut** — Ymmärrä miksi AI ehdotti sitä
4. **Anna palautetta** — Hyväksyminen/hylkääminen parantaa ehdotuksia
5. **Aloita korkeasta luottamuksesta** — Hyväksy selvät vastaavuudet ensin
