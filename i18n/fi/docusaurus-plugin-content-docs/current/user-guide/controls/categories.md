# Kontrollikategoriat

Järjestä kontrollisi loogisiin kategorioihin hallinnan helpottamiseksi.

## Miksi kategorisoida?

Kategoriat auttavat sinua:

- Löytämään kontrollit nopeasti
- Raportoimaan kategorioittain
- Määrittämään omistajuuden alueittain
- Seuraamaan kattavuutta aihealueittain

## Oletuskategoriat

Gover tarjoaa yleisiä kategoriarakenteita:

### Aihealueittain

- Tietoturva
- Tietosuoja ja tietojen suojaus
- Liiketoiminnan jatkuvuus
- Fyysinen turvallisuus
- Henkilöstöhallinto

### Tyypeittäin

- Tekniset kontrollit
- Hallinnolliset kontrollit
- Fyysiset kontrollit
- Havainnoivat kontrollit
- Ennaltaehkäisevät kontrollit

## Kategorioiden hallinta

### Luo kategoria

1. Mene kohtaan **Kontrollit** → **Asetukset** → **Kategoriat**
2. Klikkaa **Lisää kategoria**
3. Syötä:
   - Kategorian nimi
   - Kuvaus
   - Yläkategoria (valinnainen)
   - Väri (valinnainen)
4. Tallenna

### Muokkaa kategorioita

1. Mene kohtaan **Kontrollit** → **Asetukset** → **Kategoriat**
2. Klikkaa kategoriaa
3. Muokkaa tietoja
4. Tallenna muutokset

### Järjestä kategoriat uudelleen

Raahaa ja pudota kategorioita järjestääksesi ne uudelleen sivupalkissa ja luetteloissa.

### Poista kategoriat

1. Valitse kategoria
2. Klikkaa **Poista**
3. Valitse mitä tehdään kategorian kontrolleille:
   - Siirrä toiseen kategoriaan
   - Jätä kategorisoimattomaksi

## Kategoriahierarkia

Luo sisäkkäisiä kategorioita yksityiskohtaista järjestelyä varten:

```
Tietoturvakontrollit
├── Pääsynhallinta
│   ├── Todentaminen
│   └── Valtuutus
├── Verkkoturvallisuus
│   ├── Palomuurit
│   └── Valvonta
└── Tietoturva
    ├── Salaus
    └── Varmuuskopiointi
```

## Kontrollien osoittaminen kategorioihin

### Yksittäinen kontrolli

1. Avaa kontrolli
2. Muokkaa **Kategoria**-kenttää
3. Valitse alasvetovalikosta
4. Tallenna

### Massamääritys

1. Valitse useita kontrolleja
2. Klikkaa **Massatoiminnot** → **Aseta kategoria**
3. Valitse kategoria
4. Sovella

## Kategoriaraportit

Luo raportteja kategorioittain:

- Kontrollimäärä per kategoria
- Implementoinnin tila kategorioittain
- Yhdistämiskattavuus kategorioittain

## Parhaat käytännöt

1. **Pidä yksinkertaisena** — Älä ylikategorisoi
2. **Ole johdonmukainen** — Käytä samaa rakennetta koko organisaatiossa
3. **Linjaa viitekehysten kanssa** — Huomioi viitekehysrakenteet kategorisoidessa
4. **Tarkista säännöllisesti** — Kategoriat saattavat vaatia säätöä kasvaessasi
