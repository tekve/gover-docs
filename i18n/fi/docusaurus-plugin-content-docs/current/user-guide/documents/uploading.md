# Dokumenttien lataaminen

Lisää tiedostoja Gover-asiakirjakirjastoosi.

## Tuetut formaatit

| Formaatti            | Päätteet            |
| -------------------- | ------------------- |
| **Dokumentit**       | PDF, DOC, DOCX, ODT |
| **Taulukkolaskenta** | XLS, XLSX, CSV      |
| **Esitykset**        | PPT, PPTX           |
| **Kuvat**            | PNG, JPG, GIF       |
| **Teksti**           | TXT, MD             |

## Latausmenetelmät

### Yksittäinen lataus

1. Siirry kohtaan **Dokumentit**
2. Klikkaa **Lisää dokumentti**
3. Valitse **Lataa tiedosto**
4. Valitse tiedostosi
5. Täytä metatiedot:
   - Otsikko
   - Kuvaus
   - Kategoria
6. Klikkaa **Lataa**

### Raahaa ja pudota

1. Avaa Dokumentit-sivu
2. Raahaa tiedostoja tietokoneeltasi
3. Pudota latausalueelle
4. Täytä metatiedot kullekin tiedostolle

### Massalataus

1. Klikkaa **Lisää dokumentti** → **Massalataus**
2. Valitse useita tiedostoja
3. Tarkista ja muokkaa metatietoja
4. Vahvista lataus

## Dokumentin metatiedot

Ladatessa, anna:

| Kenttä         | Pakollinen | Kuvaus                            |
| -------------- | ---------- | --------------------------------- |
| **Otsikko**    | Kyllä      | Dokumentin nimi                   |
| **Kuvaus**     | Ei         | Sisällön yhteenveto               |
| **Kategoria**  | Ei         | Organisaation mukainen ryhmittely |
| **Tunnisteet** | Ei         | Haettavat tagit                   |
| **Omistaja**   | Ei         | Vastuuhenkilö                     |

## Tiedostokokorajoitukset

- Maksimitiedostokoko: 50 MB
- Suuremmille tiedostoille harkitse linkittämistä ulkoiseen tallennustilaan

## Versionhallinta

Lataa uusia versioita olemassa olevista dokumenteista:

1. Avaa dokumentti
2. Klikkaa **Lataa uusi versio**
3. Valitse päivitetty tiedosto
4. Lisää versiomuistiinpanot
5. Vahvista

Aiemmat versiot säilytetään historiassa.

## Parhaat käytännöt

1. **Käytä selkeitä nimiä** — Kuvaavat otsikot helpottavat löydettävyyttä
2. **Lisää kuvauksia** — Tiivistä dokumentin sisältö
3. **Kategorisoi johdonmukaisesti** — Käytä standardikategorioita
4. **Tagita asianmukaisesti** — Lisää relevantteja tageja hakua varten
5. **Versioi oikein** — Lataa uusia versioita, älä luo kopioita
