# Riskirekisteri

Riskirekisteri on keskitetty varasto kaikille tunnistetuille riskeille.

## Riskien tarkastelu

Siirry kohtaan **Riskit** nähdäksesi riskirekisterisi.

### Luettelonäkymä

- Lajiteltavat sarakkeet
- Suodatettavissa tilan, kategorian ja tason mukaan
- Pikatoiminnot

### Korttinäkymä

- Visuaaliset riskikortit
- Tilaindikaattorit
- Pikaesikatselu

## Riskien lisääminen

### Manuaalinen syöttö

1. Klikkaa **Lisää riski**
2. Täytä riskin tiedot:
   - **Otsikko** — Selkeä riskikuvaus
   - **Kuvaus** — Yksityiskohtainen selitys
   - **Kategoria** — Riskikategoria
   - **Omistaja** — Vastuuhenkilö
3. Tallenna riski

<!-- TODO: Add GIF showing risk creation -->

### Mallipohjista

1. Mene kohtaan **Riskit** → **Mallipohjat**
2. Selaa riskimalleja
3. Klikkaa **Käytä mallipohjaa**
4. Mukauta ja tallenna

### Massatuonti

1. Mene kohtaan **Riskit** → **Tuo**
2. Lataa mallipohja
3. Täytä riskit
4. Lataa ja vahvista

## Riskin ominaisuudet

| Ominaisuus            | Kuvaus                            |
| --------------------- | --------------------------------- |
| **Otsikko**           | Riskin nimi/kuvaus                |
| **Kuvaus**            | Yksityiskohtainen riskiselitys    |
| **Kategoria**         | Organisaation mukainen ryhmittely |
| **Omistaja**          | Vastuuhenkilö                     |
| **Tila**              | Nykyinen käsittelyn tila          |
| **Riskitaso**         | Laskettu arvioinneista            |
| **Mukautetut kentät** | Lisäseurantakentät                |

## Riskikategoriat

Järjestä riskit kategorioittain:

- Strategiset riskit
- Operatiiviset riskit
- Taloudelliset riskit
- Vaatimustenmukaisuusriskit
- Teknologiariskit
- Maineriskit

## Riskin tila

| Tila             | Kuvaus                                |
| ---------------- | ------------------------------------- |
| **Tunnistettu**  | Riski dokumentoitu, ei vielä arvioitu |
| **Arvioitu**     | Riski on arvioitu                     |
| **Käsittelyssä** | Hallintakeinoja implementoidaan           |
| **Seurannassa**  | Riski hyväksytty ja seurannassa       |
| **Suljettu**     | Riski ei enää relevantti              |

## Riskien suodattaminen

Suodata riskirekisteriäsi:

- **Riskitaso** — Kriittinen, Korkea, Keskitaso, Matala
- **Kategoria** — Riskikategoria
- **Omistaja** — Määritetty omistaja
- **Tila** — Käsittelyn tila
- **Tunnisteet** — Mukautetut tagit

## Riskin tiedot

Klikkaa riskiä nähdäksesi:

### Yleiskatsaus-välilehti

- Riskin kuvaus
- Nykyinen riskitaso
- Omistaja ja tila

### Arvioinnit-välilehti

- Arviointihistoria
- Nykyiset pisteet
- Trendi ajan myötä

### Hallintakeinot-välilehti

- Liitetyt hallintakeinot
- Hallintakeinoen tehokkuus

### Tehtävät-välilehti

- Liittyvät tehtävät
- Käsittelytoimenpiteet

### Historia-välilehti

- Muutosloki
- Tarkastuspolku

## Parhaat käytännöt

1. **Ole täsmällinen** — Kuvaa selkeästi mitä voisi mennä pieleen
2. **Määritä omistajat** — Jokainen riski tarvitsee vastuuhenkilön
3. **Arvioi säännöllisesti** — Riskitasot muuttuvat ajan myötä
4. **Liitä hallintakeinoihin** — Näytä miten riskejä lievennetään
5. **Tarkista ajoittain** — Arkistoi riskit, jotka eivät ole enää relevantteja
