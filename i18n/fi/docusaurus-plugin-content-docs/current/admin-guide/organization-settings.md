# Organisaatioasetukset

Määritä organisaatiosi asetukset Goverissa.

## Organisaatioasetuksiin siirtyminen

1. Klikkaa profiilikuvakettasi
2. Valitse **Organisaatioasetukset**

Tai siirry asetuksiin suoraan sivupalkista.

## Organisaation tiedot

### Perustiedot

- **Organisaation nimi** — Yrityksesi nimi
- **Logo** — Organisaation logo (näytetään käyttöliittymässä)
- **Kuvaus** — Lyhyt kuvaus

### Yhteystiedot

- Ensisijainen yhteyssähköposti
- Tukiyhteyshenkilö

## Käyttäjähallinta

### Käyttäjien tarkastelu

Näe kaikki organisaatiosi käyttäjät:

- Nimi ja sähköposti
- Rooli
- Tila (aktiivinen/kutsuttu)
- Viimeisin kirjautuminen

### Käyttäjien kutsuminen

1. Mene **Käyttäjät** -välilehdelle
2. Klikkaa **Kutsu käyttäjä**
3. Syötä sähköpostiosoite
4. Valitse rooli
5. Valitse työtilaoikeudet
6. Lähetä kutsu

### Käyttäjäroolit

| Rooli               | Oikeudet                     |
| ------------------- | ---------------------------- |
| **Org Admin**       | Täydet organisaatio-oikeudet |
| **Workspace Admin** | Täydet työtilaoikeudet       |
| **Editor**          | Sisällön luonti ja muokkaus  |
| **Viewer**          | Vain lukuoikeus              |

### Käyttöoikeuksien hallinta

1. Klikkaa käyttäjää
2. Muokkaa:
   - Roolia
   - Työtilaoikeuksia
   - Tilaa
3. Tallenna muutokset

### Käyttäjien poistaminen

1. Valitse käyttäjä
2. Klikkaa **Poista**
3. Vahvista poisto

## Työtilat

### Työtilojen tarkastelu

Näe kaikki organisaatiosi työtilat.

### Työtilojen luominen

1. Mene **Työtilat** -välilehdelle
2. Klikkaa **Luo työtila**
3. Syötä:
   - Nimi
   - Kuvaus
4. Tallenna

### Työtilan asetukset

Klikkaa työtilaa määrittääksesi:

- Nimen ja kuvauksen
- Käyttäjäoikeudet
- Oletusasetukset

## Tietoturva-asetukset

### Tunnistautuminen

- SSO-määritykset
- MFA-vaatimukset

### Istuntoasetukset

- Istunnon aikakatkaisu
- Samanaikaiset istunnot

## Laskutus & Tilaus

Tarkastele ja hallitse tilaustasi:

- Nykyinen sopimus
- Käyttötilastot
- Laskutushistoria
- Päivitysvaihtoehdot
