# Työtilan asetukset

Määritä yksittäisten työtilojen asetukset.

## Työtilan asetuksiin siirtyminen

1. Valitse työtila
2. Klikkaa sivupalkista **Asetukset**
3. Tai mene kohtaan **Työtilan asetukset**

## Yleiset asetukset

### Työtilan tiedot

- **Nimi** — Työtilan näyttönimi
- **Kuvaus** — Työtilan tarkoitus
- **Logo** — Työtilakohtainen logo (valinnainen)

### Työtilan valinnat

- Oletuskieli
- Päivämäärämuoto
- Aikavyöhyke

## Käyttöoikeudet

### Työtilan käyttäjien tarkastelu

Näe käyttäjät, joilla on pääsy työtilaan:

- Nimi ja sähköposti
- Työtilarooli
- Oikeustaso

### Käyttäjien lisääminen

1. Mene kohtaan **Käyttäjät** -välilehti
2. Klikkaa **Lisää käyttäjä**
3. Valitse organisaation käyttäjistä
4. Määritä työtilarooli
5. Tallenna

### Työtilaroolit

| Rooli          | Oikeudet                          |
| -------------- | --------------------------------- |
| **Ylläpitäjä** | Täysi työtilan hallinta           |
| **Muokkaaja**  | Luo ja muokkaa kaikkea sisältöä   |
| **Avustaja**   | Luo ja muokkaa nimettyjä kohteita |
| **Lukija**     | Vain lukuoikeus                   |

### Oikeuksien poistaminen

1. Valitse käyttäjä
2. Klikkaa **Poista oikeudet**
3. Vahvista

## Moduuliasetukset

Määritä asetukset kullekin moduulille:

### Viitekehykset

- Oletusnäkymä (kortit/luettelo)
- Mukautetut kentät

### Hallintakeinot

- Kategoriat
- Mukautetut kentät
- Oletustila

### Riskit

- Arviointikentät
- Pisteytyskaavat
- Riskitasojen kartoitukset

### Dokumentit

- Kategoriat
- Mukautetut kentät

### Tehtävät

- Tilat
- Kategoriat
- Mukautetut kentät

## Mukautetut kentät

Lisää mukautettuja kenttiä lisätietojen seuraamiseksi:

1. Mene moduuliasetuksiin (esim. Hallintakeinot → Asetukset)
2. Klikkaa **Mukautetut kentät**
3. Lisää kenttä:
   - Nimi
   - Tyyppi (teksti, numero, päivämäärä, valinta)
   - Vaihtoehdot (valintakentille)
4. Tallenna

## Kategoriat

Järjestä kohteet kategorioilla:

1. Mene moduuliasetuksiin
2. Klikkaa **Kategoriat**
3. Lisää/muokkaa/poista kategorioita
4. Järjestä raahaamalla

## Tunnisteet

Hallitse tunnisteita joustavaa ryhmittelyä varten:

1. Mene moduuliasetuksiin
2. Klikkaa **Tunnisteet**
3. Luo ja hallitse tunnisteita

## Tietojen vienti

Vie työtilan tiedot:

1. Mene kohtaan **Asetukset** → **Vienti**
2. Valitse vietävät tiedot
3. Valitse muoto
4. Lataa
