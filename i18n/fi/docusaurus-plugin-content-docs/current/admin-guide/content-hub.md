# Sisältöhubi

Hallitse mallipohjia, löydä uutta sisältöä ja julkaise markkinapaikalle.

## Mikä on Sisältöhubi?

Sisältöhubi on keskitetty paikka, josta löydät:

- **Mallipohjat** — Valmiiksi rakennetut viitekehykset, hallintakeinot ja dokumentit
- **Löytäminen** — Selaa markkinapaikan sisältöä
- **Asennetut** — Sisältö, jonka olet lisännyt työtilaasi

## Sisältöhubin käyttö

Siirry kohtaan **Organisaatioasetukset** → **Sisältöhubi**

## Mallipohjat

### Mallipohjien käyttö

1. Mene kohtaan **Mallipohjat**
2. Selaa saatavilla olevia mallipohjia
3. Klikkaa **Käytä mallipohjaa**
4. Valitse kohdetyötila
5. Muokkaa tarpeen mukaan

### Mallipohjatyypit

| Tyyppi            | Kuvaus                         |
| ----------------- | ------------------------------ |
| **Viitekehykset** | Kokonaiset viitekehysrakenteet |
| **Hallintakeinot**    | Hallintakeinokirjastot             |
| **Dokumentit**    | Dokumenttimallit               |
| **Riskimallit**   | Esimääritellyt riskit          |

## Löytäminen

### Markkinapaikan selaaminen

1. Mene kohtaan **Löytäminen**
2. Selaa kategorioittain:
   - Viitekehykset
   - Hallintakeinot
   - Dokumentit
3. Käytä hakua ja suodattimia
4. Esikatsele sisältöä

### Sisällön asentaminen

1. Etsi haluamasi sisältö
2. Klikkaa **Asenna**
3. Valitse kohdetyötila
4. Sisältö lisätään työtilaasi

### Sisällön tiedot

Ennen asentamista, tarkista:

- Kuvaus
- Julkaisija
- Arviot ja arvostelut
- Mitä paketti sisältää

## Asennettu sisältö

### Asennetun sisällön tarkastelu

Näe mitä olet asentanut:

- Lähde (markkinapaikka/mallipohja)
- Asennuspäivämäärä
- Työtila

### Asennetun sisällön hallinta

Asennetusta sisällöstä tulee omaasi muokattavaksi:

- Muokkaa tarpeen mukaan
- Ei automaattisia päivityksiä
- Poista jos et tarvitse

## Parhaat käytännöt julkaisemiseen

1. **Laadukas sisältö** — Varmista, että sisältö on kattavaa ja tarkkaa
2. **Hyvät kuvaukset** — Auta muita ymmärtämään mitä paketti sisältää
3. **Sopivat tunnisteet** — Tee sisällöstä helposti löydettävää
4. **Pidä ajan tasalla** — Ylläpidä julkaistua sisältöä

## Sisältöpaketit

Niputa liittyvä sisältö yhteen:

### Paketin luominen

1. Mene kohtaan **Oma sisältö**
2. Klikkaa **Luo paketti**
3. Lisää kohteet:
   - Viitekehykset
   - Hallintakeinot
   - Dokumentit
4. Lisää paketin metatiedot
5. Julkaise

### Pakettien asentaminen

Asenna kokonaiset paketit kerralla:

1. Etsi paketti Löytäminen-osiosta
2. Klikkaa **Asenna paketti**
3. Kaikki kohteet lisätään yhdessä
