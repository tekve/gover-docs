# ISO 9001 -vaatimustenmukaisuus Goverilla

ISO 9001 on kansainvälinen standardi laadunhallintajärjestelmille (QMS).

## Yleiskatsaus

| Ominaisuus     | Arvo                     |
| -------------- | ------------------------ |
| **Koko nimi**  | ISO 9001:2015            |
| **Tyyppi**     | Kansainvälinen standardi |
| **Julkaisija** | ISO                      |
| **Fokus**      | Laadunhallinta           |

## Tarkoitus

ISO 9001 auttaa organisaatioita:

- Tuottamaan johdonmukaisesti laadukkaita tuotteita/palveluita
- Parantamaan asiakastyytyväisyyttä
- Käsittelemään riskejä ja mahdollisuuksia
- Osoittamaan vaatimustenmukaisuutta

## Keskeiset periaatteet

1. Asiakaskeskeisyys
2. Johtajuus
3. Ihmisten sitouttaminen
4. Prosessimainen toimintamalli
5. Parantaminen
6. Näyttöön perustuva päätöksenteko
7. Suhteiden hallinta

## Goverin käyttö ISO 9001:lle

### 1. Lisää viitekehys

1. Mene kohtaan **Viitekehykset** → **Lisää viitekehys**
2. Valitse mallipohjista **ISO 9001**
3. Lisää työtilaasi

### 2. Dokumentoi prosessit

Kartoita laatuprosessisi:

- Ydinprosessit
- Tukiprosessit
- Johtamisprosessit

### 3. Liitä hallintakeinot

Liitä hallintakeinot seuraaville alueille:

- Laatupolitiikat
- Prosessihallintakeinot
- Dokumenttien hallinta
- Korjaavat toimenpiteet

### 4. Seuraa suorituskykyä

Monitoroi:

- Laatutavoitteita
- Asiakastyytyväisyyttä
- Prosessien suorituskykyä
- Auditointituloksia

## Integraatio

ISO 9001 integroituu seuraavien kanssa:

- **ISO 14001** — Ympäristö
- **ISO 45001** — Työterveys ja -turvallisuus
- **ISO 27001** — Tietoturva
