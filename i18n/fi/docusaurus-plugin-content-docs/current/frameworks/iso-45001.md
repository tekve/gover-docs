# ISO 45001 -vaatimustenmukaisuus Goverilla

ISO 45001 on kansainvälinen standardi työterveys- ja työturvallisuusjohtamisjärjestelmille (TTT/OH&S).

## Yleiskatsaus

| Ominaisuus     | Arvo                        |
| -------------- | --------------------------- |
| **Koko nimi**  | ISO 45001:2018              |
| **Tyyppi**     | Kansainvälinen standardi    |
| **Julkaisija** | ISO                         |
| **Fokus**      | Työterveys ja -turvallisuus |

## Tarkoitus

ISO 45001 auttaa organisaatioita:

- Ehkäisemään työperäisiä vammoja ja sairauksia
- Tarjoamaan turvallisia ja terveellisiä työpaikkoja
- Parantamaan TTT-suorituskykyä proaktiivisesti
- Täyttämään lakisääteiset vaatimukset

## Rakenne

Perustuu ISO:n ylätason rakenteeseen:

| Kohta  | Aihe                                       |
| ------ | ------------------------------------------ |
| **4**  | Organisaation toimintaympäristö            |
| **5**  | Johtajuus ja työntekijöiden osallistuminen |
| **6**  | Suunnittelu                                |
| **7**  | Tukitoiminnot                              |
| **8**  | Toiminta                                   |
| **9**  | Suorituskyvyn arviointi                    |
| **10** | Parantaminen                               |

## Keskeiset vaatimukset

### Johtajuus (Kohta 5)

- Ylimmän johdon sitoutuminen
- TTT-politiikka
- Roolit ja vastuut
- Työntekijöiden kuuleminen ja osallistuminen

### Suunnittelu (Kohta 6)

- Vaarojen tunnistaminen
- Riskinarviointi
- Lakisääteiset vaatimukset
- TTT-tavoitteet

### Toiminta (Kohta 8)

- Toiminnan suunnittelu ja ohjaus
- Hätätilanteisiin varautuminen
- Hankinta
- Urakoitsijat

## Goverin käyttö ISO 45001:lle

### 1. Lisää viitekehys

1. Mene kohtaan **Viitekehykset** → **Lisää viitekehys**
2. Valitse mallipohjista **ISO 45001**
3. Lisää työtilaasi

### 2. Tunnista vaarat

Käytä Goverin riskienhallintaa:

1. Luo riskejä työpaikan vaaroille
2. Arvioi todennäköisyys ja vakavuus
3. Määritä riskitasot

### 3. Yhdistä kontrollit

Linkitä kontrollit seuraaville alueille:

- Turvallisuuspolitiikat ja -menettelyt
- Koulutusohjelmat
- Henkilönsuojainvaatimukset (PPE)
- Hätätilannemenettelyt

### 4. Seuraa vaatimustenmukaisuutta

Monitoroi:

- Vaatimusten kattavuutta
- Häiriöseurantaa
- Auditointihavaintoja
- Parannustoimenpiteitä

## Integraatio muihin standardeihin

ISO 45001 integroituu seuraavien kanssa:

- **ISO 9001** — Laadunhallinta
- **ISO 14001** — Ympäristöjohtaminen
- **ISO 27001** — Tietoturva

Kaikki käyttävät samaa ylätason rakennetta (High-Level Structure).
