# ISO 14001 -vaatimustenmukaisuus Goverilla

ISO 14001 on kansainvälinen standardi ympäristöjohtamisjärjestelmille.

## Yleiskatsaus

| Ominaisuus     | Arvo                     |
| -------------- | ------------------------ |
| **Koko nimi**  | ISO 14001:2015           |
| **Tyyppi**     | Kansainvälinen standardi |
| **Julkaisija** | ISO                      |
| **Fokus**      | Ympäristöjohtaminen      |

## Tarkoitus

ISO 14001 auttaa organisaatioita:

- Hallitsemaan ympäristövastuita
- Saavuttamaan ympäristötavoitteet
- Täyttämään noudattamisvelvoitteet
- Parantamaan ympäristönsuojelun tasoa

## Rakenne

| Kohta  | Aihe                            |
| ------ | ------------------------------- |
| **4**  | Organisaation toimintaympäristö |

| **7**  | Tukitoiminnot                   |
| **8**  | Toiminta                        |
| **9**  | Suorituskyvyn arviointi         |
| **10** | Parantaminen                    |

## Keskeiset vaatimukset

### Ympäristönäkökohdat

- Tunnista ympäristönäkökohdat
- Määritä merkittävät vaikutukset
- Elinkaarinäkökulma

### Noudattamisvelvoitteet

- Lakisääteiset vaatimukset
- Muut vaatimukset
- Vaatimustenmukaisuuden arviointi

### Toiminnan ohjaus

- Prosessit vaatimusten täyttämiseksi
- Hätätilanteisiin varautuminen
- Ulkoistetut prosessit

## Goverin käyttö ISO 14001:lle

### 1. Lisää viitekehys

1. Mene kohtaan **Viitekehykset** → **Lisää viitekehys**
2. Valitse mallipohjista **ISO 14001**
3. Lisää työtilaasi

### 2. Tunnista ympäristönäkökohdat

Käytä Goverin riskienhallintaa:

1. Luo riskejä ympäristönäkökohdille
2. Arvioi merkittävyys
3. Määritä tarvittavat kontrollit

### 3. Yhdistä kontrollit

Linkitä kontrollit seuraaville alueille:

- Ympäristöpolitiikat
- Toimintaohjeet
- Seurantaohjelmat
- Hätätilannetoiminta

### 4. Seuraa suorituskykyä

Monitoroi:

- Ympäristötavoitteita
- Vaatimustenmukaisuuden tilaa
- Auditointihavaintoja
- Parannustoimenpiteitä

## Integraatio

ISO 14001 integroituu seuraavien kanssa:

- **ISO 9001** — Laatu
- **ISO 45001** — Työterveys ja -turvallisuus
- **ISO 27001** — Tietoturva
